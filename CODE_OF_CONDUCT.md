# Code of Conduct

Please note that the PESTO project is released with a Contributor Code
of Conduct. By contributing to this project, you agree to abide by its
terms.

PESTO adopts the **Contributor Covenant**, version 2.1, available at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

The canonical full text and translations are maintained upstream at
<https://www.contributor-covenant.org/>.

## Reporting

Concerns about conduct in PESTO spaces (issues, pull requests,
discussions, code review) may be reported to the maintainer:

- Max Moldovan, `max.moldovan@adelaide.edu.au`

Reports are treated in confidence; the maintainer will acknowledge
receipt within five working days and follow the enforcement guidelines
in the Contributor Covenant.
